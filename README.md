# Healthcare Claims Analytics Dashboard  
**Organization:** Sunrise Health Services  
**Tools Used:** SQL (SQL Server), Power BI

---

## Project Overview
This project analyzes healthcare insurance claims to evaluate reimbursement performance and claim denial patterns.  
It helps the revenue cycle and billing teams improve financial outcomes through data-driven decision making.

---

##  Dataset Information

| Column Name | Description |
|------------|-------------|
| Claim_ID | Unique claim identifier |
| Payer | Insurance company |
| Procedure_Code | Medical CPT procedure code |
| Claim_Status | Approved or Denied |
| Denial_Reason | Reason for denial (if denied) |
| Billed_Amount | Total amount billed to insurance |
| Reimbursement_Amount | Amount reimbursed by insurance |
| Service_Date | Date of service provided |

Total Records: **2000 claims**

##  Business Impact

This analysis supports:
- Identifying strong and weak-performing insurance payers
- Improving reimbursement and reducing financial loss
- Highlighting high-value medical procedures
- Enhancing documentation and claim approval strategy
